- Full-stack SaaS starter built with [Next.js](https://nextjs.org/), [Prisma](https://www.prisma.io/), and
  [Cognito authentication](https://docs.stacktape.com/resources/user-auth-pools/).
- Deployed serverlessly using the [nextjs-web resource](https://docs.stacktape.com/compute-resources/nextjs-web/) with a
  PostgreSQL [database](https://docs.stacktape.com/resources/relational-databases/) and user authentication pool.
- Includes database migrations via deploy hooks and a working CRUD API example.
