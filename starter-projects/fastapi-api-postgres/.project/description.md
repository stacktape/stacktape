- HTTP API built with [FastAPI](https://fastapi.tiangolo.com/), the modern Python web framework.
- Runs as an ASGI [web service](https://docs.stacktape.com/compute-resources/web-services/) with a Postgres
  [relational database](https://docs.stacktape.com/resources/relational-databases/). Uses SQLAlchemy for database access
  and [uv](https://docs.astral.sh/uv/) for dependency management.
