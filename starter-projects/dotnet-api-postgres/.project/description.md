- HTTP API built with [ASP.NET Core](https://dotnet.microsoft.com/apps/aspnet) (.NET 9).
- Runs in a Docker [web service](https://docs.stacktape.com/compute-resources/web-services/) with a Postgres
  [relational database](https://docs.stacktape.com/resources/relational-databases/). Uses Entity Framework Core for
  database access.
