- A [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server for AI agents and LLMs.
- Runs serverlessly in [AWS Lambda](https://docs.stacktape.com/compute-resources/lambda-functions/) with
  [Lambda URL](https://docs.stacktape.com/compute-resources/lambda-functions/#lambda-url) and response streaming
  enabled. Pay only for requests you receive.
- Uses the
  [Streamable HTTP transport](https://modelcontextprotocol.io/specification/2025-03-26/basic/transports#streamable-http)
  for compatibility with any MCP client.
