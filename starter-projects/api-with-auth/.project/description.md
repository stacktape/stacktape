- JWT-protected API with [Cognito authentication](https://docs.stacktape.com/resources/user-auth-pools/), including
  hosted sign-up/sign-in UI.
- Runs serverlessly in [AWS Lambda](https://docs.stacktape.com/compute-resources/lambda-functions/) behind an
  [HTTP API Gateway](https://docs.stacktape.com/resources/http-api-gateways/) with a Cognito JWT authorizer.
- Includes public and protected endpoints with automatic JWT validation.
