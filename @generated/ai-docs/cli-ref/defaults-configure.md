# `stacktape defaults:configure`

Sets system-wide default arguments for Stacktape commands.

You can set defaults for:
- `region`
- `stage`
- `awsProfile`
- `projectName`
- `awsAccount`

You can also configure default executables for resolving Node.js directives.

## Usage

```bash
stacktape defaults:configure
```
