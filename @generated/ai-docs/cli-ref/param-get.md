# `stacktape param:get`

Retrieves a specified parameter from a resource in a deployed stack.

## Required Arguments

- `--region`
- `--resourceName`
- `--paramName`

## Usage

```bash
stacktape param:get --region <value> --resourceName <value> --paramName <value>
```
