# Overview and basic usage

<br />

### compile-template

Compiles your Stacktape configuration into a CloudFormation template.

By default, the template is saved to `./compiled-template.yaml`. Use the `--outFile` option to specify a different path.

<br />

<br />

{/* WRITE ONLY BELOW THIS LINE */}

{/* WRITE ONLY ABOVE THIS LINE */}

```typescript
import { Stacktape } from "stacktape";

const stacktape = new Stacktape();

stacktape.compileTemplate({
  region: "<<region>>",
  stage: "<<stage>>"
});
```
