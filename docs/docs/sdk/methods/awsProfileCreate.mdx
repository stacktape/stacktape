# Overview and basic usage

<br />

### aws-profile:create

Creates a new AWS profile on your system.

You will be prompted to enter a profile name, AWS Access Key ID, and AWS Secret Access Key. The credentials are stored
in the default AWS location in your home directory and can be used by other AWS tools.
