# Overview and basic usage

### bastion:session

Starts an interactive session with a bastion host.

Your stack must include a `bastion` resource. If you have multiple bastions, you can specify which one to connect to
with the `--bastionResource` argument. The session is established over a secure SSM connection.

For more information, refer to the
[bastion documentation](https://docs.stacktape.com/bastion-servers/#connecting-to-bastion-interactive-session).

<br />

{/* WRITE ONLY BELOW THIS LINE */}

{/* WRITE ONLY ABOVE THIS LINE */}

```bash
stacktape bastion:session --stage <<stage>> --region <<region>>
```
