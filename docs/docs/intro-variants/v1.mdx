## Ship in three steps

<Steps>
  <Step title="Install the CLI">
    <CodeBlock tabs={[{ label: 'bash', lang: 'bash', code: `npm install -g stacktape` }]} />
  </Step>
  <Step title="Describe your infrastructure">
    A complete HTTP API backed by DynamoDB — with auto-generated IAM and environment variables.
    <CodeBlock
      intellisense
      tabs={[{
        label: 'stacktape.ts',
        lang: 'typescript',
        code: `import {
  defineConfig,
  DynamoDbTable,
  HttpApiGateway,
  HttpApiIntegration,
  LambdaFunction,
  StacktapeLambdaBuildpackPackaging
} from 'stacktape';

export default defineConfig(() => {
  const api = new HttpApiGateway({});

  const usersTable = new DynamoDbTable({
    primaryKey: { partitionKey: { name: 'id', type: 'string' } }
  });

  const createUser = new LambdaFunction({
    packaging: new StacktapeLambdaBuildpackPackaging({ entryfilePath: 'src/create-user.ts' }),
    connectTo: [usersTable],
    events: [new HttpApiIntegration({ httpApiGatewayName: 'api', method: 'POST', path: '/users' })]
  });

  return { resources: { api, usersTable, createUser } };
});
`
      }]}
    />
  </Step>
  <Step title="Deploy to AWS">
    You get a live API Gateway endpoint, a DynamoDB table, and a Lambda with the table name injected and IAM scoped to exactly `GetItem` / `PutItem`.
    <CodeBlock tabs={[{ label: 'bash', lang: 'bash', code: `stacktape deploy --stage prod --region eu-west-1` }]} />
  </Step>
</Steps>
