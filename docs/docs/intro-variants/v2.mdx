One command builds every function and container, resolves dependencies, and brings the stack up — with live progress for each resource.

## Three commands to your first stack

<Steps>
  <Step title="Install Stacktape">
    Install globally with npm, or run it on demand with `npx`.
    <CodeBlock tabs={[
      { label: 'npm', lang: 'bash', code: `npm install -g stacktape` },
      { label: 'npx', lang: 'bash', code: `npx stacktape --version` }
    ]} />
  </Step>
  <Step title="Configure your infrastructure">
    Define resources in TypeScript — Stacktape wires up IAM, env vars and networking for you.
    <CodeBlock
      intellisense
      tabs={[{
        label: 'stacktape.ts',
        lang: 'typescript',
        code: `import { defineConfig, HttpApiGateway, HttpApiIntegration, LambdaFunction, StacktapeLambdaBuildpackPackaging } from 'stacktape';

export default defineConfig(() => {
  const api = new HttpApiGateway({});

  const myFunction = new LambdaFunction({
    packaging: new StacktapeLambdaBuildpackPackaging({ entryfilePath: 'src/index.ts' }),
    events: [new HttpApiIntegration({ httpApiGatewayName: 'api', method: 'GET', path: '/' })]
  });

  return { resources: { api, myFunction } };
});
`
      }]}
    />
  </Step>
  <Step title="Deploy and iterate">
    Deploy to a stage, then use `stacktape dev` to run your code locally against the real cloud resources — no redeploys.
    <CodeBlock tabs={[
      { label: 'deploy', lang: 'bash', code: `stacktape deploy --stage dev --region us-east-1` },
      { label: 'dev mode', lang: 'bash', code: `stacktape dev --stage dev --region us-east-1` }
    ]} />
  </Step>
</Steps>
