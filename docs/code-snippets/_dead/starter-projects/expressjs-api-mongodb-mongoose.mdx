## Description

Simple HTTP CRUD API implemented in typescript using Express.js. API runs in a container workload. The data is stored in
a MongoDB Atlas cluster and accessed using mongoose ORM.

## Used resources

- `http-api-gateway` - routes requests to the container workload
- `multi-container-workload` - runs the API
- `mongo-db-atlas-cluster` - stores the data

## Initialize

The following command will initialize the starter project in a selected directory.

```bash
stacktape init --project-id typescript-expressjs-api-mongodb-mongoose
```

## Deploy

You can deploy your application using a single command.

```bash
stacktape deploy --stage test --region your-region
```
