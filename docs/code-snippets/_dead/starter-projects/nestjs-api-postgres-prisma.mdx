## Description

Simple HTTP CRUD API implemented in typescript using NestJS. API runs in a container workload. Data is stored in a
Postgres database and accessed using Prisma.

## Used resources

- `http-api-gateway` - routes requests to the container workload
- `multi-container-workload` - runs the API
- `relational-database` - stores the data

## Initialize

The following command will initialize the starter project in a selected directory.

```bash
stacktape init --project-id typescript-nestjs-api-postgres-prisma
```

## Deploy

You can deploy your application using a single command.

```bash
stacktape deploy --stage test --region your-region
```
