# Choosing Database Resources

Stacktape supports multiple database types for different use casesâ€”from traditional SQL databases to NoSQL, caching, and search.

## Comparison

| Resource                | Type     | Serverless     | Best For                            |
| ----------------------- | -------- | -------------- | ----------------------------------- |
| **Relational Database** | SQL      | Aurora V2 only | Traditional apps, complex queries   |
| **DynamoDB Table**      | NoSQL    | Yes            | High scale, simple access patterns  |
| **Redis Cluster**       | Cache    | No             | Caching, sessions, real-time        |
| **MongoDB Atlas**       | Document | No             | Flexible schemas, MongoDB ecosystem |
| **Upstash Redis**       | Cache    | Yes            | Edge caching, serverless            |
| **OpenSearch**          | Search   | No             | Full-text search, analytics         |
