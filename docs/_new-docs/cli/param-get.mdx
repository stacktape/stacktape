# param:get

Retrieves a specified parameter from a resource in a deployed stack.

## Usage

```bash
stacktape param:get --region <region>
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|

| `--awsAccount` (-aa) | No | string |  |
| `--stage` (-s) | No | string |  |
| `--region` (-r) | Yes | string |  |
| `--projectName` (-prj) | No | string |  |
| `--configPath` (-cp) | No | string |  |
