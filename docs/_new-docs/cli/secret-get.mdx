# secret:get

Prints details about a specified secret to the console.

## Usage

```bash
stacktape secret:get --region <region>
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|
| `--profile` (-p) | No | string |  |
| `--logFormat` (-lf) | No | string |  |
| `--logLevel` (-ll) | No | string |  |
| `--help` (-h) | No | string |  |
| `--awsAccount` (-aa) | No | string |  |
| `--region` (-r) | Yes | string |  |
