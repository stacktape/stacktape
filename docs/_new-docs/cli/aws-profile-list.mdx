# aws-profile:list

Lists all AWS profiles configured on your system.

## Usage

```bash
stacktape aws-profile:list
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|
| `--logLevel` (-ll) | No | string |  |
| `--logFormat` (-lf) | No | string |  |
