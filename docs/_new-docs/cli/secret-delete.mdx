# secret:delete

Deletes a specified secret.

## Usage

```bash
stacktape secret:delete --region <region>
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|

| `--logFormat` (-lf) | No | string |  |
| `--logLevel` (-ll) | No | string |  |
| `--help` (-h) | No | string |  |
| `--awsAccount` (-aa) | No | string |  |
| `--region` (-r) | Yes | string |  |
