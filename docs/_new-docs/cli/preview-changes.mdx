# preview-changes

Shows a preview of the changes that will be made to your stack if you deploy the current configuration.

## Usage

```bash
stacktape preview-changes --stage <stage> --region <region>
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|
| `--profile` (-p) | No | string |  |
| `--logFormat` (-lf) | No | string |  |
| `--logLevel` (-ll) | No | string |  |
| `--help` (-h) | No | string |  |
| `--awsAccount` (-aa) | No | string |  |
| `--stage` (-s) | Yes | string |  |
| `--region` (-r) | Yes | string |  |
| `--projectName` (-prj) | No | string |  |
| `--configPath` (-cp) | No | string |  |
| `--currentWorkingDirectory` (-cwd) | No | string |  |
| `--templateId` (-ti) | No | string |  |
| `--preserveTempFiles` (-ptf) | No | string |  |
