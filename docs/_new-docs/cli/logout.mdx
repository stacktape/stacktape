# logout

Removes the Stacktape API key from the current system.

## Usage

```bash
stacktape logout
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|
| `--profile` (-p) | No | string |  |
| `--logFormat` (-lf) | No | string |  |
| `--logLevel` (-ll) | No | string |  |
| `--help` (-h) | No | string |  |
| `--awsAccount` (-aa) | No | string |  |
| `--apiKey` (-ak) | No | string |  |
