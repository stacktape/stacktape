# stack:list

Lists all stacks deployed in a specified region.

## Usage

```bash
stacktape stack:list --region <region>
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|
| `--profile` (-p) | No | string |  |
| `--logFormat` (-lf) | No | string |  |
| `--logLevel` (-ll) | No | string |  |
| `--help` (-h) | No | string |  |
| `--awsAccount` (-aa) | No | string |  |
| `--stage` (-s) | No | string |  |
| `--region` (-r) | Yes | string |  |
| `--projectName` (-prj) | No | string |  |
