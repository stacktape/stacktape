# logs

Prints logs from a specified resource to the console.

## Usage

```bash
stacktape logs --stage <stage> --region <region>
```

## Options

| Option | Required | Type | Description |
|--------|----------|------|-------------|
| `--profile` (-p) | No | string |  |
| `--logFormat` (-lf) | No | string |  |
| `--logLevel` (-ll) | No | string |  |
| `--help` (-h) | No | string |  |
| `--awsAccount` (-aa) | No | string |  |
| `--stage` (-s) | Yes | string |  |
| `--region` (-r) | Yes | string |  |
| `--projectName` (-prj) | No | string |  |
| `--configPath` (-cp) | No | string |  |
| `--currentWorkingDirectory` (-cwd) | No | string |  |
| `--templateId` (-ti) | No | string |  |
| `--resourceName` (-rn) | Yes | string |  |
| `--startTime` (-st) | No | string |  |
| `--filter` (-f) | No | string |  |
| `--raw` (-rw) | No | string |  |
